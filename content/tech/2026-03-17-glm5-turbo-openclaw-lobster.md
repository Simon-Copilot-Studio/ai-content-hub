中國智譜科技（Zhipu AI）發布了 GLM-5-Turbo 模型，這是全球首個專門為 OpenClaw 的「龍蝦」（Lobster）工作流引擎進行深度優化的通用大模型。從訓練數據到最佳化目標，都基於真實的 OpenClaw 代理工作流程構建，堪稱 AI 模型廠商搶佔代理市場的標誌性事件。

## 技術規格

| 項目 | 規格 |
|------|------|
| 上下文長度 | 200K Tokens |
| 最大輸出 | 128K Tokens |
| 核心特性 | 原生 MCP 支援、Context Caching、JSON 結構化輸出 |
| 思考模式 | 多步推理 + 自我修正機制 |

## 四大核心強化

### 1. 工具呼叫穩定性

GLM-5-Turbo 強化了外部工具調用的精準度和可靠性。原生 MCP 支援讓它能連接超過 10,000 個社群 MCP 伺服器，與 GitHub、Slack、Postgres、Google Drive 等服務深度互動。

### 2. 複雜指令分解

對多層次、長鏈路指令展現更強的理解能力，能準確識別目標、規劃步驟，支援多代理協作分工。

### 3. 定時與持久性任務

針對定時觸發、長時間運行等場景進行顯著優化，能在複雜長任務中保持執行連續性。

### 4. 高吞吐長鏈路

針對高數據吞吐量和長邏輯鏈的 Lobster 任務，提升執行效率和回應穩定性。

## 什麼是 Lobster（龍蝦）？

Lobster 是 OpenClaw 的類型化工作流運行時，核心概念是把多步驟工具呼叫打包成單次確定性操作：

- **一次呼叫代替多次**：傳統上 LLM 需要不斷來回呼叫工具，每次消耗 token；Lobster 一次呼叫就能執行整個工作流
- **內建審批機制**：發送郵件或修改資料等操作，會在執行前暫停等待人工批准
- **可恢復執行**：提供 resumeToken，批准後即可繼續，大幅提升效率

## 殺手級定價：比周董演唱會票還難搶

智譜同步推出「龍蝦套餐」：

| 方案 | 價格 | Token 額度 |
|------|------|-----------|
| 基礎版 | 39 元人民幣（≈170 台幣）| 3,500 萬 tokens |
| 進階版 | 99 元人民幣（≈430 台幣）| 1 億 tokens |

這個定價極具侵略性。對比 Anthropic Claude 或 OpenAI 的 API 費率，便宜了一個數量級以上。目前處於每天開賣就秒殺的狀態，搶購難度堪比周杰倫演唱會門票。

## 產業意義

GLM-5-Turbo 的發布代表了一個重要趨勢：**AI 模型廠商開始針對特定代理框架進行客製化優化**。這不僅是模型層的競爭，更是生態系的圈地戰。

當 Anthropic 和 OpenAI 還在推通用型模型時，智譜直接瞄準 OpenClaw 用戶群，用超低價格 + 專屬優化搶市場。這個策略對中國 AI 企業來說很聰明——與其在通用模型上硬拼歐美大廠，不如在垂直場景上建立先發優勢。

值得注意的是，GLM-5-Turbo 目前尚未開源。對於重視資料安全的企業用戶，這是一個需要權衡的因素。但以這個價格和專屬優化程度，它對個人開發者和中小企業的吸引力是巨大的。
