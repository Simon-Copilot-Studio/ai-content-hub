# X → Blog Autopilot

## 流程
1. 用 Camoufox 爬 X.com trending / search 相關主題
2. 用 web_search 補充資料
3. 撰寫 SEO 優化繁體中文 Blog 文章
4. 產生封面圖
5. git push 到 blog repo
6. 發一則 X 貼文推廣

## 主題範圍
- AI 科技（LLM, agents, chips, robotics）
- 經濟財金（Fed, 台股, 美股, crypto）
- 科技產業（Apple, Google, Meta, TSMC, Nvidia）
- 與 blog 既有分類相關的任何熱門話題

## 排程
每 3 小時一次，24/7 不間斷
